การเพิ่มข้อมูลรายละเอียดสถานประกอบการ
1. เลือกเมนู
2. ระบบจะแสดงหน้าจอ ดังภาพ ให้ท่านกรอกเลขที่นิติบุคคล 13 หลัก จากนั้นกดปุ่ม Go..! เพื่อเข้าสู่ขั้นตอนต่อไป
3. ระบบจะแสดงหน้าต่าง ดังภาพ เพื่อแจ้งเตือนว่าท่านควรมีข้อมูลบันทึกอยู่ในระบบการยื่นคำขออนุญาตประกอบธุรกิจก่อนมีการเพิ่มข้อมูลในระบบนี้
4. ท่านต้องกรอกรายละเอียดข้อมูลทั้ง 3 ส่วน ได้แก่
     1) ข้อมูลสถานประกอบการ
     2) ข้อมูลผู้บริหาร และ
     3) ข้อมูลสาขา ดังภาพ
หมายเหตุ
ส่วนที่ 2) และ 3) ท่านสามารถเพิ่มหรือลบข้อมูลได้
- ถ้าต้องการเพิ่มข้อมูล กดเครื่องหมาย (ทางซ้ายมือ)
- ถ้าต้องการลบข้อมูล กดเครื่องหมาย (ทางซ้ายมือ)
5. เมื่อท่านกรอกรายละเอียดทั้งสามส่วนเรียบร้อยแล้ว กรุณาตรวจสอบความถูกต้องครบถ้วนอีกครั้ง หากครบถูกต้องถ้วนแล้ว ให้ท่านกด Save ระบบจะแสดงหน้าต่างขึ้นอัตโนมัติ ดังภาพ เพื่อเป็นการยืนยันว่าการบันทึกข้อมูลของท่านบันทึกสำเร็จแล้ว
6. ข้อมูลที่ท่านบันทึก จะปรากฎในเมนู “รายการ”
